Interface DatabaseSettings

DatabaseSettings specifies a set of settings to control the operation of the Database. Implementations should provide reasonable defaults.

child: Datastore<any>

The primary datastore. It is used to partition out "sub-domains" for collections. By default, this will be a LevelDatastore.

dispatcher

dispatcher: Dispatcher
  • Defined in database/src/db.ts:66

Dispatcher is used to dispatch local events from producers to reducers (collections). This option should generally be left undefined to use defaults.
